## Summary

Viewing the product detail page for any product with zero customer reviews crashes the application with a **500 Internal Server Error**.

---

## Steps to Reproduce

1. Open http://localhost:5000
2. Click **"View Details"** on the **USB-C Hub** product
3. Observe 500 error page

---

## Expected Behavior

The product detail page should load successfully and display a message like **"No reviews yet."**

---

## Actual Behavior

```
Internal Server Error

ZeroDivisionError: division by zero

File "app.py", line 55, in product_detail
    avg_rating = sum(product["reviews"]) / len(product["reviews"])
```

---

## Root Cause

In `app.py`, the `product_detail` function calculates the average rating with:

```python
avg_rating = sum(product["reviews"]) / len(product["reviews"])
```

When `product["reviews"]` is an empty list, `len(product["reviews"])` is `0`, causing a `ZeroDivisionError`.

---

## Suggested Fix

```python
avg_rating = (
    sum(product["reviews"]) / len(product["reviews"])
    if product["reviews"]
    else None
)
```

And in the template, handle `avg_rating == None` by showing **"No reviews yet."**
